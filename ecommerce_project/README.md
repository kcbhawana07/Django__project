# Service Booking System

## Features
- Service listing
- Service detail page
- Booking system
- Date & time slot selection
- Prevent double booking
- Booking cancellation
- User history page
- Admin panel management

## Setup Instructions

1. Clone repository
2. Create virtual environment
3. Install requirements
4. Run migrations
5. Create superuser
6. Run server
